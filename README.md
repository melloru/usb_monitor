# usb_monitor

## Описание

Проект мониторит usb порты, достает информация о флешке и CSP контейнере

## Инструменты

- [go-udev](https://pkg.go.dev/github.com/jochenvg/go-udev)

## Требования

- Go 1.24 или выше

## Установка

1. Клонируйте репозиторий: `git clone https://github.com/melloru/usb_monitor.git`


2. Установите зависимости `go mod download`
